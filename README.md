# bingo
the simple admin web application engine  
一个简单的管理web应用开发引擎实现
# 冰果
一个简单的基于元模型的web后台管理应用开发引擎。  
适合熟悉后台开发或脚本开发人员快速开发一个全功能的管理应用。
## 特性
* 支持输入参数的检查
* 支持LUA脚本作为主逻辑处理
* 具有统一的桌面管理界面
* 支持权限控制
* 支持流程控制
* 表单元模型描述，自动生成交互界面


## 引入的技术
* bingo_dao 数据元模型库
* bingo_mvc 双引擎的mvc框架
* goperlua，基于go实现的lua脚本vm
